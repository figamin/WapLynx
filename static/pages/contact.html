how to get in touch the best way to send a message to wapchan is to contact the site's email directly at admin@wapchan.org. this ensures that i'll directly get the message.
another option is to use IRC. usually some staff will be online, so if you need assistance that's a good way to go about it.
a slower and public option is to make a meta thread. this is good if you want to get some other user's feedback on whatever you're asking.
