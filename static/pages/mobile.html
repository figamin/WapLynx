Installation Guide:

16chan can now be used through the Dashchan Android app!

Steps for setting up 16chan on Dashchan
  1. Open this page on your Android device
  2. Install the Dashchan.apk - Dashchan client
  3. Install the DashchanWebm.apk - Dashchan video player libraries
  4. Install the Dashchan-16chan.apk - Dashchan Extension for 16chan
